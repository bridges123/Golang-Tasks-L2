Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Программа создает канал ch для передачи целых чисел и запускает анонимную горутину, которая отправляет числа от 0 до 9 в канал. 
Затем программа выведет числа от 0 до 9. Однако, поскольку горутина, которая отправляет значения в канал, не закрывает канал, программа выведет deadlock после прочтения всех значений из канала.
```
