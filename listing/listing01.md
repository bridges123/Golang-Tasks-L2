Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
Объявление var b []int = a[1:4] создает слайс b, который включает элементы массива a с индексами от 1 до 4 (не включительно). 
Вывод: [77 78 79]
```
